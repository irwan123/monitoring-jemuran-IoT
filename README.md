# monitoring-jemuran-IoT
Arduino Project with Blynk Android App
# Installation
- Blynk Android App on PlayStore
- Software Arduino IDE
# Component
- Node Mcu ESP8266
- Sensor cahaya (LDR)
- Sensor hujan
- Sensor ultrasonik
- Motor stepper
- 2 handphone 
- Power Bank

a.	Pin motor stepper
- IN1 = 02
- IN2 = 03
- IN3 = 04
- IN4 =  05
- --    =  G
- ++   =  UU

b.	Pin sensor cahaya
D0 = 01
GND = G
Vcc = 3V

c.	Pin sensor hujan
D0 = 0
GND = G
Vcc = 3V

d.	Pin Blynk
Button 1 = V1 (push)
Button 2 = V2 (push)
LCD = V5 - Advanced
